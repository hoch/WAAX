WXIDE: mini IDE for WAAX (revision 4)

This editor is a part of the WAAX(Web Audio API eXtension) project. WAAX is a JavaScript library designed for music and audio programming on Chrome. This small editor is powered by jQuery, Bootstrap , Code Mirror, Esprima, and Bootswatch.
