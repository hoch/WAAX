WAAX.LAB: Code Playground for WAAX (revision 5)

This editor is a part of the WAAX(Web Audio API eXtension) project. WAAX is a JavaScript library for music and audio programming on Blink-based web browser. This small editor is powered by jQuery, Bootstrap , Code Mirror, Esprima, and Bootswatch.
